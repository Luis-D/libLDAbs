To compile using CMAKE:
    - CREATE A FOLDER
    - execute: cd FOLDER; cmake ..; make;
    - Being in FOLDER, install with: make install;
    - Documentation can me compiled with: make docs;


Compile options:
    See CMakeList for a list of recommended options.


To uninstall:
    - Check install_manifest.txt

MinWG:
    -There should be an installed CMAKE and MAKE version for MinGW
    -For static libraries maybe an STATIC instruction should be added
